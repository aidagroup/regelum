# Dubuisson 2019 Fig. 1 Topology Mapping

This reconstruction follows the electrical topology in Fig. 1:

```text
Wind Turbine
  -> PMSG
  -> diode rectifier
  -> DC/DC boost converter
  -> common DC bus

Battery
  -> bidirectional DC/DC buck-boost converter
  -> common DC bus

Dump load
  -> common DC bus

Common DC bus
  -> voltage source inverter
  -> filter
  -> delta-star transformer
  -> PCC / AC load bus

Diesel engine
  -> synchronous generator + exciter/AVR
  -> PCC / AC load bus

Water treatment loads
  -> PCC / AC load bus
```

Code mapping in `examples/dubuisson2019_simulink_reconstruction.py`:

| Fig. 1 element | Regelum node |
| --- | --- |
| Wind turbine + PMSG + diode rectifier + boost MPPT | `WindTurbinePmsgAveraged` |
| Battery | `BatteryThevenin` |
| DC/DC buck-boost converter | `BatteryDcDcConverter` |
| Common DC bus capacitor | `DcLinkCapacitor` |
| Dump load | `PowerBalanceReference.dump_load_power_kw` |
| Inverter | `VoltageSourceInverter` current controller + averaged bridge |
| Filter + transformer | `LcFilterTransformer` with `i_Labc` and `v_Cabc` states |
| PCC / AC bus | `PccBus` |
| Diesel engine + synchronous generator + AVR/governor | `DieselSynchronousGeneratorReduced` |
| Water treatment load | `WaterTreatmentLoad` |

Important modeling boundary: the diesel generator is connected only to the PCC/AC
bus. The DC-link equation receives diesel/load effects only through the inverter
power, matching Fig. 1 topology.

The final scheme must include the control equations shown on pages 3 and 4, not
only the Fig. 1 power topology.

| Fig. 8 | Bode validation target for transfer function, PI and closed loop | documented; not plotted yet |

Current implementation status:

- Fig. 1 electrical topology is explicit.
- Eq. (8)-(16) are represented in the active averaged nodes or formula helpers.
- Eq. (1)-(7) are called inside `VoltageSourceInverter` every simulation step.
  The dq controller now drives averaged phase-voltage modulation
  (`m_a/m_b/m_c -> v_a/v_b/v_c`) instead of a direct active-power bridge.
- `LcFilterTransformer` integrates the filter inductor currents and capacitor
  voltages. The remaining reduced piece is the transformer/load/DG power
  coupling, which still uses an averaged power estimate for balance reporting.
- The DC-link includes damping and under/over-voltage current limiting to stand
  in for unpublished ESR, converter saturation and protection blocks present in
  a practical Simulink power-electronics model.
